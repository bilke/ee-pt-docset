Configuration¶

Radio Buttons has one setting, “Radio Button Options”, a textarea in which you specify each radio option you want as part of the group.

List one option per line:

Herbie Hancock
Bob Dylan
The Bad Plus

You can optionally specify each option’s value separately from its label:

herbie : Herbie Hancock
bobby : Bob Dylan
tbp : The Bad Plus
