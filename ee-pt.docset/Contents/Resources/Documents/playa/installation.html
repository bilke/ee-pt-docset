Installing and Updating¶

Requirements¶

Make sure your system meets the minimum requirements:

  • ExpressionEngine 2.4 or later

Installing Playa¶

  1. Upload the ee2/third_party/playa folder to system/expressionengine/third_party/
  2. Upload the themes/third_party/playa folder to themes/third_party/
  3. Install the fieldtype in Add-Ons → Fieldtypes
  4. While installing, choose to install the module and extension as well
  5. Ensure that Extensions are enabled in Add-Ons → Extensions

Updating Playa¶

  1. Upload the ee2/third_party/playa folder to system/expressionengine/third_party/, overwriting the old one
  2. Upload the themes/third_party/playa folder to themes/third_party/, overwriting the old one
  3. Go to Add-Ons → Fieldtypes → Playa to trigger Playa’s update script.
  4. Clear your browser’s cache
